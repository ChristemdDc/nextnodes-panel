3. Datos que podemos recopilar

3.1 Datos de cuenta

  • correo electrónico

  • hash de contraseña
  • identificadores internos de cuenta

3.2 Datos de autenticación externa

Si el usuario inicia sesión con Google u otro proveedor, podemos recibir identificadores y datos básicos autorizados por dicho proveedor.

3.3 Datos técnicos y de seguridad

  • IP
  • user-agent

  • identificadores de sesión o datos técnicos equivalentes

3.4 Datos operativos

  • identificadores de servidor vinculados

Usamos los datos para:

  • crear y administrar cuentas;
  • autenticar usuarios;
  • verificar correos;
  • operar el panel;
  • vincular servidores;
  • prevenir fraude, bots y accesos no autorizados;
  • mantener, depurar y mejorar el servicio;
  • cumplir obligaciones legales.

5. Base legal

La base legal podrá incluir ejecución contractual, interés legítimo, cumplimiento de obligaciones legales o consentimiento, según la jurisdicción aplicable.

6. Verificación y antiabuso

Podemos enviar códigos de verificación, enlaces temporales y aplicar controles de seguridad como límites de intentos, bloqueos temporales o medidas anti-bot.

7. Cookies y tecnologías similares

Podemos utilizar cookies, almacenamiento local y tecnologías similares para sesiones, seguridad, preferencias y funcionamiento del sitio.

8. Compartición de datos

Los datos podrán compartirse con proveedores de hosting, correo, autenticación, seguridad, analítica o autoridades cuando exista base legal suficiente. No vendemos datos personales.

9. Transferencias internacionales

Según la infraestructura utilizada, los datos podrán ser tratados fuera del país del usuario.

10. Conservación

Conservamos los datos durante el tiempo necesario para prestar el servicio, mantener la seguridad, cumplir obligaciones legales y resolver disputas.

11. Seguridad

Aplicamos medidas técnicas y organizativas razonables para proteger la información, aunque no puede garantizarse seguridad absoluta.

12. Derechos del usuario

Cuando corresponda legalmente, el usuario podrá solicitar acceso, rectificación, supresión, actualización, limitación, oposición o portabilidad.

13. Menores de edad

Términos y Condiciones de NextNodes

1. Aceptación

Al acceder o utilizar el panel web, sitio o servicios de NextNodes, el usuario acepta estos Términos y Condiciones.

2. Descripción del servicio

NextNodes es una plataforma para administrar permisos, grupos, rangos, configuraciones, vínculos de servidor y funciones relacionadas.

3. Cuentas

El usuario debe proporcionar información veraz y mantener la confidencialidad de sus credenciales. El titular del servicio podrá suspender cuentas por abuso, fraude, automatización no autorizada o incumplimiento.

4. Verificación de correo

El servicio puede exigir validación de correo mediante código, enlace temporal u otro mecanismo equivalente.

5. Inicio de sesión con terceros

El servicio puede ofrecer autenticación mediante Google u otros proveedores. El uso de dichos proveedores también queda sujeto a sus propios términos y políticas.

6. Seguridad

El usuario debe proteger sus credenciales y notificar accesos sospechosos. El servicio puede aplicar controles como rate limiting, bloqueo temporal o reautenticación.

7. Vinculación de servidores

Las claves de vinculación o tokens de emparejamiento pueden ser de un solo uso, temporales, revocables y sujetas a medidas de seguridad adicionales.

8. Uso permitido

  • automatizar accesos sin autorización;
  • interferir con cuentas, servidores o paneles de terceros;

9. Datos del usuario

10. Disponibilidad

12. Servicios de terceros

14. Terminación
